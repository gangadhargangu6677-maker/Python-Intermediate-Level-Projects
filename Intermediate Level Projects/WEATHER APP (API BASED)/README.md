PROJECT 5: 
Weather App Using API
This project introduces working with real APIs, which is a key skill for building modern applications.
1. working with APIs

2. using the requests module
 
3. handling errors

4. parsing JSON responses

5. environment variables for API keys

STEP 1: CREATE AN ACCOUNT FOR API KEY
Go to the OpenWeather website and create a free account.
Then get your API key from the dashboard.

STEP 2: INSTALL REQUESTS MODULE
Run this command in terminal:

pip install requests


STEP 3: CREATE A NEW FILE
Name it weather_app.py

Title: Weather App Using API
Description: A Python command-line application that fetches real-time weather data from OpenWeather API.
Features:

Real-time weather

Temperature in Celsius

Humidity and condition

API-based interaction
How to run:
python weather_app.py
